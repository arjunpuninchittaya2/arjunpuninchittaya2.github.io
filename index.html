2026 EV-C Quick Rules Reference
  • Run Score = 100 + Distance Score + Time Score + Bonuses + Run Penalties (low score wins).
  • Distance Score = 2.0 pt/cm × Vehicle Distance (distance from Measurement Point to Target Point).
  • Time Score = absolute value of (Target Time − Run Time).
  • Can Bonus = -0.5 × (110 − Inside Can Distance) when teams attempt and complete the can path.
  • Run Penalties: Competition Violation = +150, Construction Violation = +300.
  • Bonus Line is halfway between Start and Target. Two upright cans are placed on the Bonus Line.
  • Outside can is fixed by the Event Supervisor; inside can is competitor set between 0.0 and 100.0 cm from the outside can’s inside edge.
  • Can Bonus requires the full vehicle to travel between cans during the run.
